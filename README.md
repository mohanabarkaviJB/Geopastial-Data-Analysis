# Geospatial Data Analysis

## Objective
Analyze location-based sales data to identify suitable areas for business expansion.

## Tools Used
- Python
- Pandas
- Matplotlib
- Folium
- Google Colab

## Process
1. Loaded sales dataset.
2. Visualized sales using a bar chart.
3. Created a geographical map using latitude and longitude.
4. Identified high-demand regions.
5. Recommended locations for business expansion.

## Results
Mumbai, Delhi and Bangalore showed higher sales compared to other cities and are recommended for future expansion.

## Conclusion
Geospatial analysis helps organizations make data-driven decisions by combining sales and location information.
